# A-Bayesian-Hierarchical-Approach-to-Modelling-Non-Residential-Energy-Consumption-in-the-Maltese-Islands
This repository contains the code and full outputs associated with my undergraduate dissertation entitled "A Bayesian Hierarchical Approach to Modelling Non-Residential Energy Consumption in the Maltese Islands".
Specifically, tt includes the R scripts for the descriptive analysis of the data, for the data pre-processing and for each model presented in the dissertation. Moreover, the outputs that are not presented in the Appendix are also presented in this repository.
